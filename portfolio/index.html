Narnia: Social Media for Fashion (iOS/Android)

Repo link: →

Technical stack:

  • React
  • React Native
  • NodeJS
  • Express
  • MySQL
  • Xcode (for builds)

Narnia is a project aimed at creating a rich social media platform designed for fashion. It has all of the usual social functions such as likes, tags, and posts, but it also has some unique features like being able to design an outfit and share it, or save a piece of clothing to a personal profile. It utilized React Native thoroughly and was designed for both iOS and Android. Narnia was made up of a team of 4 with I acting as Scrum master.

Snips: Code Snippet Manager (Web)

Repo link: →

Technical stack:

  • Angular
  • NodeJS
  • Express
  • MySQL

Snips is a web code manager designed for speed and ease-of-use. With a single click, you can save a code snippet and with elasticsearch, every keypress brings up results instantaneously. The UI is made to be lightweight and focused for work.
